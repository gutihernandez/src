What just Happened and Why?

To understand why there is a difference between these two queries, one must know what reasoning means. Step2 is sending a query to a database which has a specific ontology and has reasoning feature. Step3 is sending a query to a database which has no feaure of reaosining while using the same ontology as Step2's database. It can be clearly seen that, query gave us different results. Reason behind is as follows;

  • There is an instance of owl:Thing created and called isThisAlgida.
  • isThisAlgida sells Magnum
  • Magnum is type of IceCream
  • IceCreamCompany equivalent to sells min 1 IceCream
  • That is how the reasoner understood that isThisAlgida is an IceCreamCompany.
  • With reasoner one can achieve explicit results

    • What happened on the second query?

  • Well second query only got BenJerry because it is told that BenJerry is a IceCreamCompany
  • Since second database has its reasoning OFF, it just gave us the results what we said to it.
